# 🐾 Práctica de Programación Orientada a Objetos en Python  
## Clases Abstractas y Polimorfismo (Animales)

## 📌 Descripción

Este proyecto implementa el uso de clases abstractas en Python mediante el módulo `abc`.  
Se define una clase base abstracta llamada `Animal`, la cual establece un método obligatorio que debe ser implementado por todas las clases derivadas.  

A partir de esta clase, se crean implementaciones específicas como `Perro` y `Gato`, donde cada una define su propio comportamiento al momento de "hablar".

---

## 🧱 Estructura del Proyecto

Clases implementadas:

- `Animal` (Clase abstracta)
- `Perro`
- `Gato`
